Objetivos Específicos del OVI del Curso:

  • Identificar las Vulnerabilidades de un Sistema Informático.
  • Conocer los Diferentes Tipos de Amenazas que Pueden Afectar un Sistema Informático.
  • Aprender a Proteger el Sistema Informático para así mismo Proteger la Información.
